## Quaternion Algebra ℍ

The quaternion algebra ℍ has basis {1, i, j, k} with multiplication rules:

```
i² = j² = k² = -1
ij = k,  jk = i,  ki = j
ji = -k, kj = -i, ik = -j
```

ℍ is a **non-commutative** division algebra over ℝ.

Key subspaces:
- **Re(ℍ)** = span{1} — real part
- **Im(ℍ)** = span{i, j, k} — imaginary part (dim = 3)

---

## Biquaternion Algebra ℂ⊗ℍ

A biquaternion has the form:

```
q = (a + ib₁)·1 + (b + ib₂)·i + (c + ic₂)·j + (d + id₂)·k
  = z₀ + z₁i + z₂j + z₃k,   z_μ ∈ ℂ
```

**Key isomorphism** (proved [L0]):

```
ℂ⊗ℍ  ≅  Mat(2, ℂ)
```

This is the standard result connecting biquaternions to 2×2 complex matrices.
Explicit isomorphism:

```
1 ↦ I₂,   i ↦ iσ₁,   j ↦ iσ₂,   k ↦ iσ₃
```

where σ₁, σ₂, σ₃ are the Pauli matrices.

## Automorphism Group

```
Aut(ℂ⊗ℍ) ≅ [GL(2,ℂ) × GL(2,ℂ)] / ℤ₂
```

Left GL(2,ℂ) action → SU(2)_L × U(1)_Y after restriction to unitary elements  
Right GL(2,ℂ) action → U(1)_EM phase

## Z₂×Z₂×Z₂ Involution Structure

The imaginary subspace Im(ℍ) admits three involutions:

```
P_I : q  →  -iqī  (flip j,k components)
P_J : q  →  -jqj̄  (flip i,k components)
P_K : q  →  -kqk̄  (flip i,j components)
```

These satisfy P_I² = P_J² = P_K² = 1 and form a Z₂×Z₂×Z₂ group.
The Lie algebra generated by their commutators is 𝔰𝔲(3).

## Norm and Inner Product

The standard quaternionic norm:

```
|q|² = qq̄ = a² + b² + c² + d²  ∈  ℝ
```

For biquaternions, the norm is complex-valued in general.
