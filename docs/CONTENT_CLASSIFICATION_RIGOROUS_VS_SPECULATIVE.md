# UBT Content Classification: Rigorous vs. Speculative

## Purpose

This document provides a complete classification of UBT content by scientific rigor level, enabling researchers to:
1. Distinguish established results from exploratory hypotheses
2. Identify which claims are supported by derivation vs. conjecture
3. Guide peer review and publication strategy
4. Maintain scientific honesty and credibility

---

## Classification Levels

### Level 1: RIGOROUS (Mathematically Derived)
- Based on established mathematics
- Reduces to known physics in appropriate limits
- Internally consistent
- No free parameters beyond Standard Model inputs

### Level 2: ESTABLISHED (Physically Grounded)
- Consistent with experimental data
- Based on standard field theory techniques
- May include empirical parameters
- Testable in principle

### Level 3: EXPLORATORY (Conceptual Framework)
- Mathematical framework developed
- Physical interpretation proposed
- Parameters not yet determined
- Testability uncertain

### Level 4: SPECULATIVE (Hypothesis)
- Conceptual proposal only
- Limited mathematical development
- No experimental predictions
- Requires substantial future work

## Recommended Usage by Context

### For Academic Papers (Peer-Reviewed Journals):
**Include**: Levels 1-2 only
- Appendices: A, R, P1, P3, C, D, K, QG
- Mention Level 3 content as "future directions"
- Omit Level 4 content entirely

### For Conference Presentations:
**Include**: Levels 1-3
- Core results in main presentation
- Exploratory ideas in "ongoing work" section
- Brief mention of speculative extensions in Q&A

### For Preprint Archives (arXiv, OSF):
**Include**: Levels 1-4 with clear labeling
- Comprehensive version with all appendices
- Explicit warnings on speculative content
- Cross-reference this classification document

### For Popular Science / Outreach:
**Include**: Conceptual overview of all levels
- Emphasize what is established vs. speculative
- Use analogies and visualizations
- Link to technical documents for details

## Quality Assurance Checklist

Before claiming any result publicly:

### For Level 1 (RIGOROUS) Claims:
- [ ] Mathematical derivation complete and error-free
- [ ] Reduces to known physics in limits
- [ ] Independent verification by collaborators
- [ ] Consistency checks performed

### For Level 2 (ESTABLISHED) Claims:
- [ ] Consistent with experimental data
- [ ] Parameters explicitly stated as empirical inputs
- [ ] Compared with Standard Model predictions
- [ ] Error estimates provided

### For Level 3 (EXPLORATORY) Claims:
- [ ] Framework mathematically consistent
- [ ] Open questions clearly identified
- [ ] Path to testability outlined
- [ ] Not presented as established result

### For Level 4 (SPECULATIVE) Claims:
- [ ] Clearly labeled as speculation or hypothesis
- [ ] Ethical implications considered (especially consciousness)
- [ ] Not used for medical/life decisions
- [ ] Presented as long-term research direction
