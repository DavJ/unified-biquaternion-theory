
## Hubble Tension as Synchronization Latency

UBT interprets the Hubble tension as global synchronization overhead
in a discrete-time multi-branch cosmology.

Parameters:
- F = 256
- N = 16
- O ≈ 16–21 ticks
- δ ≈ 0.08

Relation:
H0_local ≈ H0_global / (1 − δ)

See appendix_hubble_latency.md and calibrate_hubble_latency.py
